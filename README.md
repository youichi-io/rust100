# rust100 🦀

Rustの基礎力を鍛えるための **Rust 100本ノック** リポジトリです。  
小さな問題を数多く解くことで、文法・標準ライブラリ・考え方を身につけることを目的としています。


## 目的

- Rustの基本文法と標準ライブラリに慣れる
- 「書いて・考えて・振り返る」を繰り返す
- 小さな問題を通して実践力を積み上げる

各問題は **独立した Cargo プロジェクト**として構成されています。


## 構成

- 問題は `README.md`
- 自分の解答は `src/main.rs`
- 模範解答は `src/bin/ans.rs`
- 解説・考え方は `explain.md`

という形で、**問題・解答・解説を分離**しています。


## 問題一覧

👉 問題はこちらから  
➡ **[problems/](problems)**

（各ディレクトリを開くと、その問題の README が表示されます）


## 実行方法

例：no001 を実行する場合

```bash
cd problems/no001

```

## 自分の解答を実行

```bash
cargo run

```

## 模範解答を実行

```bash
cargo run --bin ans

```
